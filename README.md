## Install

npm:
```
npm i morphogen
```

yarn:
```
yarn add morphogen
```

### Configurating

- Create __morphogen directory inside of your project (or use --morf_dir for custom dir)
- Create templates and scenarios (see examples dir)
- Run morphogen


## Run examples

```
node lib/cli.js -d examples/scenario
```
